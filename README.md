# TRAOPD1 – VALORANT Match Tracker

A desktop application for tracking live VALORANT matches, player stats, match history, and weapon skins.

Built with **Electron**, **React**, **Vite**, and **Tailwind CSS**.

## Features

- **Live Match Tracking** – Real-time 5v5 player stats during Agent Select and in-game
- **Player Database** – Save players with categories (friend / toxic / avoid) and notes
- **Match History** – Browse past matches with expandable scoreboard details
- **Weapon Skins Display** – See Phantom, Vandal, Operator, and Melee skins per player
- **Rank System** – Color-coded tiers from Iron to Radiant
- **True Stretch** – Resolution presets for competitive setups (4:3, 5:4, 16:10)
- **Settings** – Overlay opacity, hotkeys, and alert configuration

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- VALORANT must be installed (Windows only – reads local Riot Client data)

### Install

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Tests

```bash
npm test
```
